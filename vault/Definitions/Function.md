
## Definition

A `function` is a mapping from [[set]] $X$ to [[set]] $Y$:

$f: X \rightarrow Y$

Assigns each `element` of $X$ to exactly one `element` of $Y$

Think:

$f: \mathbb{R} \rightarrow \mathbb{R}$

- The `domain` of the function (from $X$) is R
- The `codomain` of the function (to $Y$) is R

if $f(x) = x^2$

The `range` = `image` = $[0, \infty)$

## Function

A rule that transforms inputs into outputs

## Why it Matters

[[Operation]]s are special cases of `functions`

- Every operation is a function
- But, not every function is an operation

## Flash Cards

- What is another term for **function**?
	- **map**
- What is a **function**?
	- A **function** from **set** X to **set** Y  
	- Assigns each element of X to exactly **one element of Y**
- What is an **even** function and even polynomial
	- [$]f(-x) = f(x)[/$]  
	- [$]z^2, z^4, ...[/$]
- What is an odd function and an odd polynomial?
	- [$]f(-x) = -f(x)[/$]  
	- [$]z, z^3, z^5, ...[/$]
- What is the **domain** of a **function**?
	- The **set** of **allowed** input values to a function
- Formalize a function (formal)
	- [$]f: \mathbb{F} \rightarrow \mathbb{F}, f(x) = x^2[/$]  
	- Domain: [$]\mathbb{R}[/$]  
	- Codomain: [$]\mathbb{R}[/$]  
	- Range (image): [$][0,\infty)[/$]
- What components can functions be decomposed into?
	- Into an **even** component and an **odd** component
	- [$]f(x) = f_e(x) + f_o(x)[/$], where: 
	- [$]f_e = \frac{f(x) + f(-x)}{2}[/$] 
	- [$]f_o = \frac{f(x) - f(-x)}{2}[/$]
- Unpack what "continuous real-valued function on the interval [0,1]" means
	- Written as [$]\mathbb{R}^[0,1][/$] or [$]f: [0,1] \rightarrow \mathbb{R}[/$]  
		- on the interval [0,1] means the functions **domain** = [0,1]  
		- real valued function means **codomain** = [$]\mathbb{R}[/$]  
		- continuous = no jumps/breaks
- What does f(x) mean (simple - dont overthink)
	- The value of function f, at value x
- What is the **codomain** of a function?
	- The **set** that the **outputs** are declared to live in
- What is the **image** of a **function**?
	- The range (image is more precise) 
	- The set of all outputs [$]f(x)[/$] for all x in the domain
- - What is the **range** of a function? What is it also called?
	- The set of "reachable" outputs of the function - the **image** [$]\operatorname{range} \subseteq \operatorname{codomain}[/$]
