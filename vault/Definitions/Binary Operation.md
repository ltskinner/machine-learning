
## Definition

A `binary operation` on a set S is a function:

$\oplus : S \times S \rightarrow S$

that takes two elements of S and returns another element of S

## Intuition

A rule for combining two elements into one element, sourced from the same set

## Why it Matters

Binary operations define `algebraic structure`, including:

- [[Group]]
- [[Ring]]
- [[Field]]
- [[Vector Space]]

All algebra begins with a set + a `binary operation`

## Examples

- Addition on $\mathbb{R}$
- Multiplication on $\mathbb{C}$
- [[Matrix Multiplication]]

## Flash Cards


- What is a **binary operation** on a set S? (Formal)
	- A function:  $\oplus: S \times S \rightarrow S$ that takes **two elements** of S and returns **another** element of S
- What key requirement distinguishes a valid **binary operation**?
	- **Closure** - the result must stay inside the same set
- Why is **closure** required for a **binary operation** to be **well-defined**?
	- Without closure, the operation can produce elements **outside the set**, breaking algebraic structure
- Example of a **binary operation** and what it acts on (3)
	- Addition on [$]\mathbb{R}[/$]  
	- Multiplication on [$]\mathbb{C}[/$]  
	- Matrix multiplication on matrices
