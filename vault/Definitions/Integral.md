## Flash Cards

- What is [$]\int_{1}^{0} f = b [/$] equivalent to w/ [$]dx[/$] notation?
	- [$]\int_{1}^{0} f(x) dx = b [/$], or to express a constraint [$]\int_{1}^{0} f(x) dx - b = 0 [/$]
- Take the integral of [$]\int z^2 dz[/$] note **NO** integral bounds. What is this asking?
	- What function is this the derivative of? [$]\int z^2 dz = \frac{1}{3}z^3 + C[/$]
- Integrating an even function over [$][-a, a][/$] yields?
	- [$]\int_{-a}^{a} z^3 dx \equiv 2 \int_{0}^{a} z^3 dx[/$] will double from 0 to [$]a[/$]
- Integrating an odd function over [$][-a, a][/$] yields?
	- [$]0 \equiv \int_{-a}^{a} z^3 dx[/$]
- EVALUATE the integral [$]\int_{-1}^{1} z^2 dz[/$]
	- [$]\int_{-1}^{1} z^2 dz = (\frac{1}{3}(1)^3 + C) - (\frac{1}{3}(-1)^3 + C)[/$] 
	- [$]= \frac{1}{3} - (- \frac{1}{3}) = \frac{2}{3}[/$] (note C cancels on evaluation)
