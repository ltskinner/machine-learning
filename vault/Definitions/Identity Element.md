
An element e is an `identity` if:

$\alpha \oplus e = \alpha$ for all $\alpha$ in the set

## Examples

- Additive identity: 0
	- $\lambda + 0 = \lambda$ for all $\lambda \in \mathbb{C}$
- Multiplicative identity: 1
	- $\lambda 1 = \lambda$ for all $\lambda \in \mathbb{C}$

## Intuition

A value that **does nothing** under the operation

### Why it Matters

Defines neutrality and enables inverses
