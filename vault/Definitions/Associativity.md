
## Definition

An operation $\oplus$ is `associative` if:

$(\alpha \oplus \beta) \oplus \lambda = \alpha \oplus (\beta \oplus \lambda)$

## Examples

$(\alpha + \beta) + \lambda = \alpha + (\beta + \lambda)$

$(\alpha\beta)\lambda = \alpha(\beta\lambda)$

## Intuition

Grouping does not change the result

### Why it Matters

Allows chaining operations without parentheses

### Failure Cases

[[Subtraction]] and [[Division]] are NOT associative
