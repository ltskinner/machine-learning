
The **function** $T: V \rightarrow W$ is surjective if its **range** equals W
- $\operatorname{range} T = W$

If T is surjective, then the W must have a basis that is [[span]]ned by the map on V (and vice versa):
$T(v_1, ..., v_n) = w_1, ..., w_m$ so $\operatorname{range} T = span(T(v_1), ..., T(v_n)) = span(w_1, ..., w_m) = W$

And, if T is surjective, then $\exists T(v_i) = w_i$ for  $i = 1, ..., m$

`Surjective` = `onto` 

Linear maps to **higher**-dimensional spaces are NOT surjective
- If dim V < dim W, then no **linear map** from V to W is **surjective**
##### Surjective Example (NOT surjective)

- NOT $T(v_1, v_2) = (0, w_2)$
- YES $T(v_1, v_2) = (w_1, w_2)$
