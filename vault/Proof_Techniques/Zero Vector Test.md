
Used to prove whether a subspace or not

If a set is a subspace, it must contain the zero vector:

1. Define the zero vector in $F^n$ (0, 0, ..., 0)
2. Inject into an equation, taking any of the terms of the test coordinate as its own linear equation
3. See if anything non-zero comes out the other end

## Example

$S = \{(x1, x2) | x + 2y = 3 \}$

Zero vector: (0, 0)

Check: 0 + 2(0) = 3

0 = 3, which is False
